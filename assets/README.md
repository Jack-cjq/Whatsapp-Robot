# Assets 目录

此目录包含应用程序的静态资源文件。

## 需要的文件

### 图标文件
- `icon.ico` - 应用程序图标 (Windows)
- `icon.png` - 应用程序图标 (PNG格式)
- `icon.icns` - 应用程序图标 (macOS)

### 其他资源
- 背景图片
- 按钮图标
- 其他UI元素

## 图标规格

### Windows (.ico)
- 尺寸: 256x256 像素
- 格式: ICO
- 用途: 应用程序图标、任务栏图标

### PNG格式
- 尺寸: 512x512 像素
- 格式: PNG
- 用途: 通用图标

## 获取图标

您可以从以下来源获取图标：
1. 设计工具 (如 Figma, Adobe Illustrator)
2. 图标库 (如 Flaticon, Icons8)
3. 在线图标生成器

## 注意事项

- 确保图标文件具有适当的版权许可
- 图标应该清晰可辨，在不同尺寸下都表现良好
- 建议使用矢量格式作为源文件，然后导出为所需格式
