### 🚀 启动方式

#### 1. 演示模式（推荐）
```bash
npm run start:demo
```
- 只启动界面，不连接WhatsApp
- 适合查看界面功能和测试
- 无网络依赖，启动快速

#### 2. 完整模式
```bash
npm start
```
- 启动完整的WhatsApp机器人
- 需要Chrome浏览器
- 需要扫描WhatsApp二维码

#### 3. 开发模式
```bash
npm run dev
```
- 启动完整功能并打开开发者工具
- 适合开发和调试

### 🎯 使用步骤

#### 首次使用
1. **启动演示模式**：
   ```bash
   npm run start:demo
   ```

2. **查看界面功能**：
   - 仪表板：查看系统状态
   - 日志：查看操作记录
   - 配置：设置管理员和参数
   - 帮助：查看使用说明

3. **配置管理员**：
   - 在配置页面设置管理员用户名
   - 保存配置

#### 完整功能使用
1. **启动完整模式**：
   ```bash
   npm start
   ```

2. **扫描二维码**：
   - 使用手机WhatsApp扫描显示的二维码
   - 等待连接成功

3. **在群组中使用**：
   - 发送"查账"查看余额
   - 发送"+100"增加金额
   - 发送"帮助"查看所有命令
